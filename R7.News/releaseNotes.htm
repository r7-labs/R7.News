Version 1.3.0

End-user changes:

  • Implemented discuss functionality (#27) - DNN Forum and ActiveForums supported out of the box.
  • Fixed GetNewsEntriesByTerms_FirstPage results is unreliable.
  • Fixed "Show more" button is visible even if all available news entries are displayed (#26).
  • Fixed DescriptionTermUrlProvider inproperly detects URLs in term description (#31).
  • Module name in search results/settings cannot be customized via SearchableModules.resx (#32).

Developer changes:

  • Added xUnit-based tests project.
  • Updated packaging scripts.
  • Switch to model binding for all controls.
  • Simplified CSS rules.
  • Various code refactorings based on BCH analysis.

Version 1.0.0

This is the initial release of R7.News subsystem, featuring two main modules: R7.News.Stream used to display list of aggregated news, filtered by taxonomy terms and weight filters; and R7.News.Agent used do display and manage news, associated with the page.
